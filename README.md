# Vista AI News Magazine

این مخزن نقطهٔ شروع پیاده‌سازی «مجلهٔ خبری هوش مصنوعی ویستا» است که در سند مشخصات محصول توصیف شده است. این نسخهٔ ابتدایی اسکَفلد Next.js 14، پیکربندی تایپ‌اسکریپت، TailwindCSS، Prisma و اسکریپت Seed اولیه را فراهم می‌کند تا مسیر توسعهٔ ویژگی‌های پیچیده‌تر (جمع‌آوری خبر، ترجمهٔ خودکار، پنل ادمین و …) هموار شود.

## ساختار پروژه

```
root
├─ src/app/             # App Router و صفحات محلی‌سازی‌شده
├─ src/components/      # کامپوننت‌های UI (در آینده)
├─ src/lib/             # ماژول‌های کمکی (DB، i18n، env)
├─ prisma/              # schema.prisma و seed.ts
├─ public/              # دارایی‌های استاتیک (در آینده)
├─ .husky/              # هوک‌های Git
├─ package.json         # اسکریپت‌ها و وابستگی‌ها (pnpm)
└─ tailwind.config.ts   # پیکربندی TailwindCSS
```

## پیش‌نیازها

- Node.js 18 یا جدیدتر
- pnpm 8 یا جدیدتر (در فایل `package.json` به نسخهٔ توصیه‌شده اشاره شده است)
- پایگاه‌دادهٔ Postgres در دسترس

## راه‌اندازی محلی

```bash
pnpm install
pnpm prisma migrate dev --name init
pnpm prisma db seed
pnpm dev
```

سپس برنامه روی آدرس `http://localhost:3000` قابل مشاهده است. مسیر ریشه به صورت خودکار به `/fa` هدایت می‌شود و نسخهٔ انگلیسی در `/en` در دسترس است.

## اسکریپت Seed

اجرای `pnpm prisma db seed` یک حساب مدیر (admin@vista-ai.ir) با گذرواژهٔ پیش‌فرض `Strong!Pass#123` ایجاد می‌کند و دسته‌بندی‌ها، برچسب‌ها و چند منبع خبری whitelisted را مقداردهی اولیه می‌کند. توصیه می‌شود پس از ورود اولین بار، گذرواژه را تغییر دهید.

## مراحل بعدی پیشنهادی

- پیاده‌سازی احراز هویت Supabase و حفاظت از مسیر `/admin`.
- افزودن رابط کاربری shadcn/ui و اجزای Tailwind برای صفحات عمومی و پنل مدیریت.
- ساخت ماژول جمع‌آوری خبر (`src/jobs/fetch-news.ts`) با RSS Parser، متاسکریپر و ترجمهٔ خودکار.
- اتصال سیستم اعلان و مانیتورینگ مطابق با سند اصلی.
- تکمیل تست‌های واحد برای توابع کمکی و پردازش محتوا.

کد پروژه تحت مجوز اختصاصی شرکت شبکه هوشمند ابتکار ویستا توسعه می‌یابد؛ برای استفادهٔ مجدد حتماً هماهنگ شود.
